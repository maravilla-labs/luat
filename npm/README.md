# @maravilla-labs/luat

Svelte-inspired server-side Lua templating CLI.

## Installation

```bash
npm install -g @maravilla-labs/luat
```

## Usage

```bash
# Create a new project
luat init my-app

# Start development server with live reload
luat dev

# Build for production
luat build
```
